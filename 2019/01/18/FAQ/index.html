装blog过程中遇到的问题

发表于 2019-01-18 | 分类于 FAQ | 阅读次数：

.git文件夹不显示可能是隐藏了

传文件 git init

语言改不了中文：
\themes\next\languages下 zh-CN.yml文件改成名字为zh-Hans.yml

ssh-keygen -t rsa -C "1411601520@qq.com" c要大写

hexo安装
npm install -g hexo-cli
判断是否安装
npm list hexo-deployer-git

HEXO出现ERROR Deployer not found
git的问题:
deploy的type 的github需要改成gitnpm
install hexo-deployer-git –save

scheme 更换本地预览ok 上传到GitHub无效:
依次执行
hexo clean
hexo g
hexo d
还可能是要把所有打开的文件都关闭

重新打开cmd hexo 显示 Usage: hexo < command >
cd 到博客名字的那层目录
npm install hexo-server --save
